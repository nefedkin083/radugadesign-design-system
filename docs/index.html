Inter
Резервный шрифт. Body-текст в плотных интерфейсах, мелкие подписи, табличные данные. Не использовать в заголовках и постерах.

Основной шрифт — Geologica. Везде, где это возможно — в Google Slides, на сайтах, в email-шаблонах, в документации, в постерах.

Geologica — нейтральный гротеск с пропорциями, близкими к Pragmatica Next, но открытый по лицензии OFL и доступный одной строкой в Google Slides и Google Docs. Это значит: ни один документ, который мы передаём наружу, не теряет шрифт при открытии у партнёра. Slides не подменяет его на Arial при экспорте.

Запасной — Inter. Только для плотных интерфейсов, мелких подписей и табличных данных, где Geologica может казаться слишком «бренд-голосовой». Не использовать в заголовках и постерах.

Чего не делаем. Не используем Pragmatica Next в материалах, которые передаются наружу — у партнёра шрифта нет, ляжет на Times. Внутренние Figma-файлы — оставляем как есть.

Weights

Базовые: 400 для body, 500 для подзаголовков, 600 для заголовков, 300 для крупных дисплейных надписей с разреженным эффектом.

Подключение

Google Slides / Google Docs. Insert → Font → More fonts → ищем «Geologica» → Add. После этого ставим как Theme font на весь шаблон.

Web. Через Google Fonts:

<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Geologica:wght@300;400;500;600;700;800&display=swap" rel="stylesheet">

font-family: 'Geologica', sans-serif;

Self-hosting. Файлы — github.com/google/fonts/tree/main/ofl/geologica. Variable TTF: Geologica[CRSV,SHRP,slnt,wght].ttf. Лицензия OFL — можно класть в репозиторий.

У Radugadesign нет одного фирменного цвета. Цитата из guideline: «есть цветовая палитра, которую можно использовать при работе с материалами, где она необходима — например, в 3D-графике, иллюстрациях или мерче». Это намеренное решение бренда: цвет — инструмент, а не идентичность.

Идентичность держится на типографике (Geologica) и принципах композиции. Цвет — поверх.

Структура палитры

13 hue-колонок × 6 tone-рядов = 78 цветов. Колонки идут по спектру: cyan → sky → blue → indigo → purple → magenta → pink → red → coral → orange → yellow → lime → green. Тоны — от насыщенно-светлого (bright) до угольно-тёмного (charcoal).

Как выбирать цвет

Для одного материала (постер, обложка, инсталляция) — одна-две колонки палитры. Не смешиваем три и больше hue в одной композиции, иначе теряется системность.

Контрастные пары — между крайними тонами одной колонки: bright поверх charcoal той же колонки или наоборот. Это гарантирует AA-контраст для текста.

Соседние колонки (cyan + sky, orange + yellow) — для мягких градиентов и переливов. Колонки с противоположных концов (cyan + red, blue + orange) — для резких контрастных сцен.

Чего не делаем

Не используем нейтральные серые из старых проектов как «бренд-цвет». В файле много серых из конкретных постеров и кейсов — это атавизмы, не палитра. Если нужен нейтральный — берём ink #000 или paper #FFF.

Не задаём один доминирующий цвет на всю коммуникацию бренда. Палитра — инструмент с переменной частью; постоянная часть — типографика и сетка.
