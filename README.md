# Radugadesign Design System

Машиночитаемая дизайн-система Radugadesign. Источник правды для людей и LLM-агентов: токены, принципы, паттерны и инструкции, по которым внешний агент может строить материалы в фирменном стиле бренда.

## Структура

- `tokens/` — атомарные значения: цвета, типографика, сетки, отступы, радиусы, тени. Формат — YAML/JSON.
- `principles/` — правила и логика стиля в markdown. Что мы делаем и почему. Что не делаем.
- `patterns/` — собранные примеры: вёрстка постера, презентации, поста, сайта. Со ссылками на токены и принципы.
- `llm/` — system-prompt и инструкции для агентов: как читать эту систему и применять её при генерации.
- `site/` — статический сайт-витрина для людей и поиск/индекс для агентов.
- `figma-snapshot/` — выгрузка исходного Figma-файла rd4-C через API. Машинный дамп, не редактируем руками.
