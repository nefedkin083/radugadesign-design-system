# Иконки

Иконографика Radugadesign зафиксирована в Figma rd4-C → страница **icons**.
Две группы: **Icons** (системные) и **Unicorn+** (брендовые).

## Icons — системный набор

Все иконки линейные, монохромные, одна толщина stroke. Стиль минималистичный, без теней и скевоморфизма.

**Навигация и указатели**
- Стрелки во всех направлениях (↑ ↓ ← → и диагонали)
- Несколько типажей буллетов (точка, тире, квадрат, галочка)
- Пин / закреп
- Ссылка

**Инструменты**
- Ручка (редактирование)
- Загрузка
- Лампочка (идея)
- FAQ

**Медиа и коммуникация**
- Аудио: наушники, микрофон, плейбар
- Камера / объект
- Глаз

**Время и организация**
- Календарь
- Колокольчик / уведомление
- Папка
- Закладка

**Реакции**
- Лайк / сердце
- Звёздочка
- Огонёк

**Информационные**
- Молния (срочно / энергия)
- Цифры (нумерация)
- Нет заряда
- Человек
- Планета

## Unicorn+ — брендовые иконки

Единорог — символ radugadesign («Unicorns of freedom since 2008»). Используется как брендовый знак, не как декоративный элемент.

**Правила применения Unicorn+:**
- В карточках карусели — можно совмещать с лого `r`
- Не использовать на каждом материале — только там, где нужна брендовая идентификация
- Размер пропорционален логотипу `r`, не крупнее

## Правила применения иконок

Иконки не смешиваются со сложными иллюстрациями в одном макете — либо иконки, либо фото/графика.

Цвет иконки: `#000` (ink) на светлом фоне или `#FFF` (paper) на тёмном. Иконки из палитры — только если цвет несёт смысловую нагрузку (например, иконка-статус).

Минимальный tap-target для интерактивных иконок — 44×44 px.
